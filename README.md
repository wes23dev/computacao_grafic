# Trabalho 1 - Animacoes de Planetas

Projeto Vite + TypeScript com duas animacoes em arquivos distintos:

1. Terra esferica com translacao e rotacao.
2. Terra plana com translacao e rotacao.

## Arquivos principais

- `src/animations/terraEsferica.ts`
- `src/animations/terraPlana.ts`
- `src/main.ts`

## Executar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
